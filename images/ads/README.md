# Ad Images

Ad creative images for Meta campaigns. Hosted here for public URL access via Pipeboard MCP.

## How It Works

1. Drop ad images in this folder
2. Push to GitHub
3. Image is publicly accessible at: `https://raw.githubusercontent.com/RexAffects/rexaffects-site/main/images/ads/FILENAME`
4. Use that URL with Pipeboard's `upload_ad_image` tool to push to Meta ad account

## Naming Convention

`YYYY-MM-DD_campaign_description.png`

Examples:
- `2026-04-10_vja_claudio-testimonial.png`
- `2026-04-10_gumroad_blob-tracking-demo.png`
- `2026-04-10_theta_spring-promo.png`
